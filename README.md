## ⚙️ Quick Start

To build and run:

```bash
# Install laravel dependencies
composer install --no-dev

# Install js modules
npm install

# Build the project
npm run build

# Add the database
php artisan migrate

# Run in dev
php artisan serve && npm run dev
```
